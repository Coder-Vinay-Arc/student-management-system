# 🎓 Student Management System

A full-stack Student Management System built using **Spring Boot**, **React**, and **MySQL**. This project performs complete CRUD (Create, Read, Update, Delete) operations through a RESTful API.

## 🚀 Tech Stack

- Spring Boot
- React (Vite)
- MySQL
- Spring Data JPA
- Axios
- Tailwind CSS

## ✨ Features

- Add Student
- View Students
- Update Student
- Delete Student
- Responsive UI
- REST API Integration

## 📁 Project Structure

```text
student-management-system
├── backend
└── frontend
```

## ▶️ Run the Project

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
